Cover GeoJSON shape with geohashes

Published: Wed 18 March 2015
By Jerry Xu

In Programming.

Geohash-shape

Transform a shapely GeoJSON Shape (Box, Polygon, MultiPolygon) to a list of geohashes that cover it.

Usage

Get an array of geohashes to cover the shape.

polygons_geojson = """{"type":"MultiPolygon","coordinates":[[[[-122.54974365234374,37.77071473849609],
        [-122.53601074218751,37.70989935485515],[-122.46597290039062,37.72184917678752],
        [-122.43301391601562,37.738141282210385],[-122.48794555664061,37.78808138412046],
        [-122.54974365234374,37.77071473849609]]],[[[-122.46734619140625,37.846663684549156],
        [-122.2723388671875,37.743571187449064],[-122.22290039062499,37.84557924966551],
        [-122.24899291992188,37.89761413273189],[-122.46734619140625,37.846663684549156]]]]}"""

polygons_geom = load_from_JSON(polygons_geojson)

geohash_shape(polygons_geom, 6)

Result

[
'9q8ye8',
'9q8ye9',
'9q8yec',
...
'9q9pd4',
'9q9pd5'
]

Options

  • shape: shapely shape to cover
  • precision: geohash precision
  • mode: defines filtering of the returned geohashes
    • intersect - all geohashes intersect the shape.
      Use 'threashold' option to specify a percent of least coverage. The 'threashold' option is a relatively expensive operation because it needs to calculate the area of each intersection.
    • inside - all geohashes inside the shape.
    • center - all geohashes whose center is inside the shape.

Installtion

pip install geohash-shape
